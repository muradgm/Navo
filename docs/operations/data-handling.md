# Data Handling

## Data Categories

### User-provided trip data

- dates
- base location
- traveler count
- child ages
- preferences
- saved activities
- custom itinerary

### External data

- weather forecasts
- official website links
- Google Maps links
- opening-hours references

### Generated data

- recommendation scores
- budget estimates
- itinerary suggestions
- AI-generated planning visuals

## Privacy Principles

- Do not collect precise location unless needed.
- Do not store children names by default.
- Child ages are enough for planning.
- Make account creation optional until sync is required.
- Keep local-only mode possible.

## Image Policy

- Local AI-generated images are acceptable as visual aids if not presented as real photos.
- Review/social images should not be embedded without rights and privacy review.
- Official images require license/permission review.
