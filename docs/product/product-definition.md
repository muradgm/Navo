# Product Definition

## One-Sentence Definition

TripOS is a personalized trip companion that creates and adapts practical travel plans based on dates, location, preferences, weather, transport, budget, and trip purpose.

## Primary Promise

> Enter the facts of your trip. TripOS tells you what is worth doing, what fits today, how to get there, what it costs, and what to avoid.

## Target Users

### 1. Family Tourist

Needs:
- kid-friendly activities
- low-stress pacing
- food planning
- weather-safe alternatives
- day bag/checklists
- emergency information
- transport simplicity

### 2. Solo / Couple Tourist

Needs:
- efficient discovery
- hidden gems
- walking-friendly plans
- food/café suggestions
- photo-worthy places
- budget control

### 3. Business Traveler

Needs:
- short-window plans
- after-meeting ideas
- dinner near hotel/client site
- airport/station timing
- low-risk options
- business-friendly places

### 4. Short-Stop Visitor

Needs:
- 2-hour, 4-hour, half-day plans
- luggage-friendly choices
- station/airport-aware routing
- must-see compression

### 5. Local Weekend Explorer

Needs:
- nearby plans
- weather-based ideas
- family activities
- free/low-cost options
- fresh discoveries

## Core Inputs

Required:
- destination/city
- trip dates
- base location or arrival point
- traveler count
- trip type

High-value optional:
- arrival/departure times
- accommodation address
- ages of kids
- food preferences
- budget range
- weather tolerance
- mobility/walking limits
- transport preference
- interests
- business schedule blocks
- must-do / avoid list

## Core Outputs

- day-by-day plan
- activity cards
- today plan
- smart filters
- budget estimate
- weather-aware recommendations
- transport notes
- grocery/essentials links
- food strategy
- emergency/practical info
- favorites
- custom itinerary builder
- conditional side-trip suggestions

## MVP Success Criteria

The MVP is real if a user can:

1. Create a trip with dates and base location.
2. Add traveler preferences and constraints.
3. See a useful activity list filtered by cost, type, weather, age fit, transport, and energy.
4. Build a day plan from selected activities.
5. See live budget estimates.
6. Get food/essentials guidance.
7. See weather-aware recommendations.
8. Use it comfortably on a phone.

## Non-MVP

- booking hotels/flights
- selling tickets
- full AI autonomous travel agency
- real-time transport routing API integration
- social sharing
- marketplace monetization
- fully global city coverage

These can come later.
