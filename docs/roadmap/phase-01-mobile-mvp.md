# Phase 01 — Mobile MVP

## Goal

Turn the Basel planner into a reusable destination-based mobile trip planner.

## Deliverables

- generic trip setup
- generic destination config
- activity cards
- filters
- day builder
- budget calculator
- weather integration
- food/essentials section
- safety section
- bilingual structure

## Acceptance

- runs locally
- mobile-first
- no broken Add/favorite buttons
- trip data survives refresh
- no hardcoded Basel-only logic in generic components
