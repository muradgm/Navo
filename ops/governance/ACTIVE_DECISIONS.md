# Active Decisions

## DEC-001: Frontend-first MVP

Decision: Build React + Vite + TypeScript frontend first before backend complexity.

Reason: Product behavior needs validation before server architecture.

## DEC-002: Weather dynamic but honest

Decision: Use dynamic weather API where available, but show unavailable state for future dates outside forecast window.

Reason: Fake weather damages trust.

## DEC-003: Local AI images allowed as planning visuals

Decision: AI-generated images can be used as local visual aids but must not be represented as real photos.

Reason: stable assets without copyright risk.
