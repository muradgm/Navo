# Product Definition

## Definition

TripOS is a mobile-first personal trip operating system for turning travel context into practical daily decisions.

## The Job

When a traveler enters trip facts and constraints, TripOS helps answer:

- what should I do today?
- what fits my time window?
- what fits my budget?
- what is good for my traveler type?
- what should I do if it rains or gets too hot?
- what is near my base?
- what should I avoid today?
- what should I keep as backup?

## MVP User

The first MVP should serve a traveler planning one trip to one destination, with a strong mobile experience.

## MVP In Scope

- destination data pack
- trip setup
- traveler preferences
- activity filters
- day builder
- dynamic weather fetch
- budget estimates
- food/essentials guidance
- safety/practical info
- local persistence

## MVP Out of Scope

- booking marketplace
- flight/hotel purchase
- paid transport API integration
- social feed
- global coverage
- account system unless needed
- monetization

## Quality Bar

The app is not acceptable if it behaves like a static blog list.

It must react to user context.
