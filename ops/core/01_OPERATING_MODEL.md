# Operating Model

## Roles

- PM: decomposes work and controls acceptance
- UX: product flow, mobile UI, interaction quality
- FS: full-stack implementation
- DATA: destination data, schemas, source registry
- AI: personalization/recommendation logic and prompts
- QA: tests, review, regression checks
- DEVOPS: deployment and environment reliability
- GTM: positioning, landing copy, validation messaging

## Rule

Agents do not self-expand scope. If they find adjacent work, they report it as follow-up.
