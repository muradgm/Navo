# Barcelona generated activity visuals

Barcelona activity cards currently use policy-safe generated SVG visuals through `apps/public/activity-image-fallback.js`.

## Safety constraints

- No real people.
- No faces.
- No children.
- No sexual, violent, hateful, political, or shocking content.
- No text baked into images.
- No fake app UI inside images.
- No copied photographs or third-party image assets.

## Scope

These visuals are original, abstract, family-friendly travel illustrations generated from the referenced Barcelona activity filenames.

They are a bridge until final reviewed WebP files are added under this folder with the same filenames referenced by the Barcelona destination pack.
